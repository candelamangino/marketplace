# Assets

Esta carpeta debe contener los siguientes archivos para que la app funcione correctamente:

- `icon.png` - Icono de la aplicación (1024x1024px)
- `splash.png` - Imagen de splash screen (1242x2436px recomendado)
- `adaptive-icon.png` - Icono adaptativo para Android (1024x1024px)
- `favicon.png` - Favicon para web (48x48px)

Para desarrollo con Expo Go, estos archivos son opcionales ya que Expo Go usa sus propios assets.

Puedes generar assets básicos usando:
```bash
npx expo install expo-asset
```

O usar el generador de assets de Expo:
https://www.favicon-generator.org/
